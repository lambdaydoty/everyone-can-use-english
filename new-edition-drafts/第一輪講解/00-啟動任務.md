# 啟動任務

第一步，不是 “教學”，不說 “方法”，沒有 “技巧”，而是 “**直接幹活**”。

## 任務要點

- 說自己想說的話
- 每天至少三小時
- 每天都要交作業

### 說自己想說的話

- 先用母語寫下差不多 100 ～ 150 字，記錄 “自己想說的話”；隨著任務的推進，逐步增加篇幅。
- 使用 AI 輔助工具將其轉換成英文表達
- 使用 AI 輔助工具將英文文字轉換成語音檔案
- 跟讀、朗讀、直至完整背誦。

實際操作中，這裡的難點可能是 “用自己的母語寫下一段話” —— 大多數人會遇到 “說不清楚的障礙”，比如，很多人可能意識不到 “自己的母語其實非常差”…… 實在寫不出來，甚至不知道該寫什麼的話，那麼，可以 “改寫別人的話”，比如，你每天聽到或者讀到的 “金句” 甚至別人發的 “微博”、“朋友圈” 什麼的。也可以是 “自己家裡每天都用的日常對話”，還可以是 “託福作文題”、“GRE 作文題” 等等。實在搞不定的時候，可以直接用有聲書的片段。

另外，有空的話，可以先去補補《李笑來的寫作課》。

AI 輔助工具，付費方案最好的是 OpenAI，無論是文字生成、還是語音生成。免費方案的話，文字生成可以用 LM Studio，甚至直接用免費的 Google Translate；語音生成可以用 MS Edge TTS engine，或者寫成 `html` 檔案之後，用 MS Edge 瀏覽器開啟，也能有朗讀功能。另外，網上也有很多方案，可以自己想辦法。

雖然最初的時候，你可能要自己折騰 AI 輔助工具，但，我們在開發 “輔助工具”，也將會持續完善，很快就會分發到社群成員手裡，任何小白都可以直接上手使用。（但，早期依然需要使用者自己擁有 OpenAI 的 API Key 才行……）

### 每天至少三小時

如果做不到每天三小時，那就算了吧 —— 當然，其實誰都可以做到的。

只有大聲開口練習的時間才算數 —— 注意，一定是 “大聲”；雖然開口了，但只不過是小聲嘟囔，不算數。

- 折騰文字的時間不算；
- 反覆聽錄音卻不開口或只是跟著嘟囔的時間不算；
- 若是還有精力，其他時間裡還可以閱讀英文書籍文章、看英文影片電影劇集，但，只要沒有大聲開口都不算……
- 隨著時間的推移，複習之前的任務，朗讀、重新錄音的時間可以計入。

雖然最好是 “一口氣三小時”，但，“累計三小時” 也可以。

三小時之外，如果你有時間精力，依然應該花時間輸入英文，無論是閱讀、還是聽有聲書或者看影片影視劇集。

### 每天都要交作業

作業不是要 “交給老師”，而是自己存檔備份。

作業內容包括：

1. 最終生成的文字和語音（一個文字檔案和一個語音檔案）
2. 自己的第一遍朗讀錄音（一個語音檔案）
3. 自己的最後一遍朗讀錄音（一個語音檔案）
4. 在這個過程中，遇到的困難（解決的、未解決的），以及想到的可能的解決方案，練習過程中的感悟。（一個文字檔案）

以上打包壓縮，以日期為檔名儲存。這些記錄，是將來 “在社群裡當老師”，“在家裡當老師”，或者起碼 “給自己當老師”、“給自家孩子當老師” 必須的資格證明。

### 簡要的補充說明

這個任務最關鍵的一點是，“沒有老師”、“沒有他人教學或者糾正”、“沒有他人監督和促進” —— 只靠自己。

最初的三個月，沒有任何技巧而言，就是 “生學硬練”……

事實上，最初 1 ～ 3 個月最重要的 “核心任務”，不是發音技巧什麼的，而是 “極大提高英文記憶力” —— 所以，每天結束的時候，都得以 “熟練背誦” 作為結束。

關於這個任務的 “教學講解”，此後會詳細深入。現在最重要的是，“直接開始幹活”。
