## v0.7.0

### 新增功能

- 增加匯入文件功能，可跟讀、翻譯等，支援匯入本地 epub, txt, markdown 等格式文字，同時支援線上文章匯入
- 重構軟體頂部欄和側邊欄
- 本地 whisper 增加更多配置項，可選擇 `whisper.cpp` 版本
- 匯入音影片時，增加是否壓縮的選項
- 增加預設的 TTS 配置
- Mixin 可以選擇掃碼登入

### 修復故障

- 修復字幕舊格式導致的錯誤
- 重複匯入相同資源時，將更新資源的修改時間，使其置於排序靠前
- 修復字幕中特殊符號（例如破折號）導致顯示異常的問題

### 其他

- 重構部分程式碼，最佳化可維護性
- 最佳化 UI 樣式
