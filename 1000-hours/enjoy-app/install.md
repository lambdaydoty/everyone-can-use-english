# 下載安裝

Enjoy App 是一個跨平臺的桌面應用，可以在 Windows、Mac 和 Linux 等電腦裝置上執行。

請根據電腦裝置的作業系統，下載相應的版本安裝使用。

當前最新版本：**v0.7.9**

## Windows

支援 Windows 10 以上版本。

[點選下載](https://dl.enjoy.bot/app/win32/x64/Enjoy-0.7.9%20Setup.exe)

下載後，雙擊即可安裝。

::: tip Windows 10 安裝報錯怎麼辦？
Windows 10 系統在安裝時可能會報錯：

```bash
A JavaScript error occurred in the main process
```

這是由於缺少某些依賴導致，可以嘗試以下步驟解決：

1. 升級系統，將作業系統升級到最新版本；
2. 下載 [vs_redist.x64.exe](https://aka.ms/vs/17/release/vc_redist.x64.exe) 安裝。

然後再嘗試安裝 Enjoy。
:::

## Mac

根據使用 Mac 電腦的晶片不同，需要下載不同的版本。

- [Silicon 晶片版本(arm64)](https://dl.enjoy.bot/app/darwin/arm64/Enjoy-0.7.9-arm64.dmg)
- [Intel 晶片版本(x64)](https://dl.enjoy.bot/app/darwin/x64/Enjoy-0.7.9-x64.dmg)

::: info 如何檢視本機配置
M1 以後的 Mac 電腦型號(M1、M2、M3)，均為 Silicon 晶片。

如果你不清楚自己電腦的具體型號，可以點選螢幕左上方的  符號，在彈出來的選單中，選擇第一個選項 “關於本機（About This Mac）”，在彈出的視窗中，即可看到本機的硬體和系統配置。

如果顯示的是 _Apple M 系列_，即為 Silicon 晶片。
:::

::: warning 警告
推薦使用 macOS 12 及以上版本，macOS 11 可能無法使用部分功能。
:::
