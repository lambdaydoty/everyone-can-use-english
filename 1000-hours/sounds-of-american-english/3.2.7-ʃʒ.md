## 3.2.7.2. 舌尖起始位置

再一次，關鍵在於發聲時舌尖的起始位置，齦脊附近 —— 即，位置 ③。

<span class="pho">ʃ</span> 和 <span class="pho">ʒ</span> 都是舌尖起始位置**接近但沒有接觸**齦脊（位置 ③）。

<span class="pho">ʃ</span> 對很多人來說也許難度並不大，因為 <span class="pho">ʃ</span> 的舌尖起始位置其實和漢語拼音的 <span class="pho">[sh]</span> 是一樣的。

然而，<span class="pho">ʒ</span> 卻可能非常難 —— 雖然它其實只不過是 <span class="pho">ʃ</span> 這個清子音的濁子音版本。舌尖起始位置相同，清子音 <span class="pho">ʃ</span> 沒有聲帶振動，而濁子音 <span class="pho">ʒ</span> 有聲帶振動。

除了英語之外，大多語言裡沒有 <span class="pho">ʒ</span> 這個子音。即便是在英語裡，含有這個子音的詞彙也很少，常見的不過幾十個而已。中國學生會不由自主地用 <span class="pho">dʒ</span> 替代 <span class="pho">ʒ</span> —— 因為 <span class="pho">dʒ</span> 是一個他們熟悉的子音，相當於中文拼音中的 <span class="pho">zh</span>。<span class="pho">ʒ</span> 和 <span class="pho">dʒ</span> 的區別在於，<span class="pho">ʒ</span> 發聲時**舌尖貼近但沒有接觸到齦脊**；而 <span class="pho">dʒ</span> 發聲時起始的舌尖貼在齦脊（與 <span class="pho">ʒ</span> 同一個位置 ③）。

少數人知道  <span class="pho">dʒ</span> 替代 <span class="pho">ʒ</span> 不是一回事，但，由於舌尖起始位置並不準確，所以他們說出來的 <span class="pho">ʒ</span> 聽起來更接近 <span class="pho">r</span> —— 舌尖縮得過頭了 —— <span class="pho">r</span> ④ 發聲時舌尖起始位置相對於 <span class="pho">ʒ</span> ③ 要更靠後一點。
