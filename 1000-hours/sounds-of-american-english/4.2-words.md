# 4.2. 單詞

一個**單詞**（*word*）至少由一個**音節**（*syllable*）構成。

## 4.2.1. 重音、次重音、非重音、弱音

重點在於，英文的音節有**重音**、**次重音**、**非重音**的區別，這一點和亞洲語言明顯不同。

如果一個單詞只有一個音節，單獨讀出的時候，就當作是**重音**（*stress*）音節讀出。而一個多音節詞彙中**有且只有一個**重音音節，但，可能還有另外一些音節是**次重音**（*secondary stress*），在音標中使用 <span class="pho">ˌ</span> 作為標記。比如，*serendipity* <span class="pho alt">ˌserənˈdɪpət̬i</span><span class="speak-word-inline" data-audio-us-male="/audios/serendipity-us-male.mp3" data-audio-us-female="/audios/serendipity-us-female.mp3"></span>，有一個**重音**和一個**次重音**。

而 *[Pneumonoultramicroscopicsilicovolcanoconiosis](https://en.wikipedia.org/wiki/Pneumonoultramicroscopicsilicovolcanoconiosis)*<span class="speak-word-inline" data-audio-us-female="/audios/En-us-pneumonoultramicroscopicsilicovolcanoconiosis.ogg.mp3"></span> 總計有 19 個音節，其中 7 個是次重音，唯一的重音是 <span class="pho">oʊ</span>……

**非重音** 音節中，還有**弱音**，即被弱化成 <span class="pho">ə</span> 的央母音（*schwa*）音節。

細細體會 *individual* <span class="pho alt">ˌɪndəˈvɪdʒuəl</span><span class="speak-word-inline" data-audio-us-male="/audios/individual-us-male.mp3" data-audio-us-female="/audios/individual-us-female.mp3"></span> 這個詞，如果每個音節都清晰地用同樣的強度讀出來的話，應該是 <span class="pho alt">ɪn-dɪ-vɪ-dʒju-əl</span>…… 這裡面，第 2 個音節 <span class="pho alt">dɪ</span>裡的母音 <span class="pho">ɪ</span> 被弱化為 <span class="pho">ə</span>；另外，第 4 個音節 <span class="pho alt">dʒu</span> 裡的母音 <span class="pho">ju</span> 也被弱化成了 <span class="pho">ə</span>，但保留了 <span class="pho">u</span> 的音色；而處於音節末尾的 <span class="pho">l</span>，之前自帶著輕微的 <span class="pho">ə</span>，於是連成了很輕的 <span class="pho">ʷᵊl</span>……

![5-syllables-individual](/images/5-syllables-individual.svg)

## 4.2.2. 強讀與弱讀

單詞的讀音實際上都有兩種形式：

> * 強讀式（***accented** form*）
> * 弱讀式（***weak** form*）

字典裡的音標標註或者真人朗讀示範，都是**強讀式**。任何單詞，在單獨說出的時候都是**強讀式**。但，在自然語流之中，不是每個單詞都是強讀式，只有少數是**強讀式**，更多是**弱讀式**。

比如 *our* 這個單詞，詞典裡標註的當然是 <span class="pho alt">ˈaʊ.ɚ</span><span class="speak-word-inline" data-audio-us-male="/audios/hour-us-male.mp3" data-audio-us-female="/audios/hour-us-female.mp3"></span> —— 這是**強讀式**，每一個單詞的 “預設形式”。然而，在實際的自然語流中，你更多聽到的並不是這個飽滿的三母音 <span class="pho alt">ˈaʊ.ɚ</span> 而是它的**弱讀式**，一個很輕很短的 <span class="pho alt">ɑr</span>：

> It's *our* pleasure.<span class="pho alt">ˈaʊ.ɚ</span><span class="speak-word-inline" data-audio-us-male="/audios/Its-our-pleasure-us-male.mp3" data-audio-us-female="/audios/Its-our-pleasure-us-female.mp3"></span>

關於**強讀式**與**弱讀式**的區別，得等到[音節](4.1-syllables)、[單詞](4.2-words)、[意群](4.3-grouping.md)、[連線](4.4-linking)全都講清楚之後，到了[句子](4.5-sentences)那個章節再深入講解。

## 4.2.3. 詞典讀音

現在，你已經知道了，所有的詞典，都沒有，也不可能標註單詞的**弱讀形式** —— 可能是因為在自然語流中，弱讀形式就不存在一個所謂的 “標準”。

除此之外，你還必須瞭解一個可能是所有學校裡的老師都沒有告訴過你的一個**重要事實**：

> 詞典中的**音標標註**，或者**真人發聲示例**，其實都只是 “**完成度極高**” 的 “**半成品**”。

首先，人類的自然語音，就是很難使用一個符號系統去完美地標註……

英文的例子一會兒再講，現在先舉一箇中文的例子。你琢磨一下自己平日裡是如何說 “不知道” 這三個字的？當然，我們可以用拼音分別標註這三個字 <span class="pho">[bù zhī dào]</span> —— 可你真的是這樣三個字分開一字一頓地說的嗎？你說出來的聲音，只能用拼音湊合著標註成 <span class="pho">[búr(i)dào]</span> —— “**不**” 你可能讀的不是四聲，更像是速度很快的二聲；“**知**” 你說得像 “日”，但速度很快，乃至於可能感覺只有一個聲母 `[r]` 沒帶著韻母 `[i]`；最後，三個字裡，只有 “**到**” `[dào]` 這個字算是字正腔圓地說完了。****

更何況，音標本身也是個 “**無法完美**” 的符號系統。

比如，<span class="pho">ə/əː</span> 的兒化音，到底是用 <span class="pho">ər/əːr</span> 好呢？還是用 <span class="pho">ɚ/ɝː</span> 更好呢？大多數情況下看不出什麼區別，但，就是有一些詞會出現無論用哪種都不太合適的情況。

比如，*literally* 這個詞。

> * [牛津詞典的標註](https://www.oxfordlearnersdictionaries.com/definition/english/literally?q=literally)是： <span class="pho alt">ˈlɪtərəli</span>，而後，真人示例中，英音 <span class="speak-word-inline" data-audio-uk-female="https://www.oxfordlearnersdictionaries.com/media/english/uk_pron/l/lit/liter/literally__gb_1.mp3"></span> 聽起來像 <span class="pho alt">ˈlɪtrəli</span>，美音像 <span class="speak-word-inline" data-audio-us-female="https://www.oxfordlearnersdictionaries.com/media/english/us_pron/l/lit/liter/literally__us_1.mp3"></span> <span class="pho alt">ˈlɪtɚəli</span>（其實還真就是這個標註才對）……
> * [劍橋詞典的標註](https://dictionary.cambridge.org/pronunciation/english/literally)是：<span class="pho alt">ˈlɪt̬.ɚ.ᵊl.i</span>，而後，真人示例中，英音 <span class="speak-word-inline" data-audio-uk-female="https://dictionary.cambridge.org/media/english/uk_pron/u/ukl/uklis/uklisp_017.mp3"></span> 聽起來像 <span class="pho alt">ˈlɪtrəli</span>，美音像 <span class="speak-word-inline" data-audio-us-male="https://dictionary.cambridge.org/media/english/us_pron/l/lit/liter/literally.mp3"></span> <span class="pho alt">ˈlɪt̬ərəli</span>……

如果你要是誤以為所有詞典裡的標註或者示範都是 “權威的”、“標準的”，那麼，沒比較過，沒深入探究倒也罷了，只要稍微認真一點，就得掉入 “無所適從” 的尷尬境地。

這個詞，本身就有好幾種不太一樣的讀法。針對其中的任何一種讀法，想要用音標完美標註都感覺相當費勁，甚至勉為其難。

又比如，另外一個相當常用的單詞，*temperature* —— 我曾經誤以為它讀作 <span class="pho alt">ˌtem.pə.ˈreɪ.tʃɚ</span> —— 它實際讀音的音標標註是 <span class="pho alt">ˈtem.pɚ.ə.tʃɚ</span>，結果呢，在日常生活中，你更常聽到的是 <span class="pho alt">ˈtem.pə.tʃɚ</span><span class="speak-word-inline" data-audio-us-male="/audios/temperature-us-male.mp3"></span>，或者是 <span class="pho alt">ˈtem.pɤtʃɚ</span>，甚至乾脆是 <span class="pho alt">ˈtem.ptʃɚ</span><span class="speak-word-inline" data-audio-us-female="/audios/temperature-us-female.mp3"></span>

再舉一個更常用的詞作為例子，*important* <span class="pho alt">ɪmˈpɔːr.tənt</span><span class="speak-word-inline" data-audio-us-male="/audios/important-us-male.mp3"></span>，你可能無論如何都想不到，這個由 3 個音節構成的單詞，你可能有 3 處細節需要注意，但，竟然從來沒意識到……

> * 第一個是最普遍的，很多人不分母音長短，所以會說成 <span class="pho alt">ɪmˈpɔ.tənt</span> 或者 <span class="pho alt">ɪmˈpɔr.tənt</span>；
> * 第二個細節是，這個詞的第一個 <span class="pho">t</span> 經常會被說成喉塞音 `ʔ`，所以，你很多時候聽到的其實是 <span class="pho alt">ɪmˈpɔːr.ʔənt</span><span class="speak-word-inline" data-audio-us-male="/audios/important-us-male.mp3"></span>；
> * 第三個細節是，這個詞的最後一個 <span class="pho">t</span> 其實是 “可選 <span class="pho">ᵗ</span>”（即，非重讀音節末尾 <span class="pho">n</span> 之後的 <span class="pho">t</span>）—— <span class="pho alt">ɪmˈpɔːr.ʔənᵗ</span> —— 那個 <span class="pho">ᵗ</span> 在自然語流中經常並不存在，比如，當你說 *It's a very important aspect.*<span class="speak-word-inline" data-audio-us-male="/audios/It-s-a-very-important-aspect-alloy.mp3" data-audio-us-female="/audios/It-s-a-very-important-aspect-nova.mp3"></span> 的時候，與後面的 <span class="pho">æ</span> 連起來讀的（示例語音中並未連讀），更可能是 <span class="pho">n</span> 而不是 <span class="pho">t</span>。

我們其實已經用了一些英文詞典裡的音標中並不存在的符號才能把一些細節講清楚，比如，阻塞音符號 <span class="pho">◌̚</span>，比如，比央母音更弱的一個音， <span class="pho">ɤ</span>，再比如喉塞音 <span class="pho">ʔ</span>…… 可即便如此，英文有那麼多的單詞（幾十萬個），有至少 7,000 個左右構成獨特的音節，於是，無論如何都不大可能用一個並不完美的音標系統完美地標註任何自然語音。
