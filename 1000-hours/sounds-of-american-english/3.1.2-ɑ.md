# 3.1.2. <span class="pho">ʌ</span>/<span class="pho">ɑː</span>/<span class="pho">ɑːr</span>

在美式英語中，<span class="pho">ɑ</span> 有 *3* 個版本，短母音 <span class="pho">ʌ</span>、半長母音 <span class="pho">ɑː</span>，和長母音的兒化版 <span class="pho">ɑːr</span>。其中，<span class="pho">ʌ</span> 的口型最小，<span class="pho">ɑː</span> 和 <span class="pho">ɑːr</span> 的口型最大。

## 3.1.2.2. 短母音 <span class="pho">ʌ</span>

在美式英語中，<span class="pho">ʌ</span> 略有 <span class="pho">ə</span> 的音色，甚至有些美國詞典乾脆用 <span class="pho">ə</span> 這個符號替代 <span class="pho">ʌ</span> —— 比如韋氏詞典（Merriam-Webster）中 *but* 這個詞的音標被寫成 <span class="pho alt">bət</span><span class="speak-word-inline" data-audio-us-female="/audios/but-us-female.mp3" data-audio-us-male="/audios/but-us-male.mp3"></span>。另外，英國人說 <span class="pho">ʌ</span> 的時候，口型相對美國人更大一些。

## 3.1.1.3. 半長母音 <span class="pho">ɑː</span>

在《劍橋英語發聲詞典》中，<span class="pho">ɑː</span> 雖然帶著一個長母音符號，它實際的長度相當於**半長母音**，即，它比短母音長一些，比長母音短一些。

另外，《劍橋英語發聲詞典》裡的 <span class="pho">ɑː</span> 還對應著英式英語中的短母音 <span class="pho">ɒ</span>。除了《劍橋英語發聲詞典》之外，大多詞典會把 *hot* 的音標標註為 <span class="pho alt">hɑt</span> —— 即，沒有長母音符號。但是《劍橋英語發聲詞典》會標註成 <span class="pho alt">hɑːt</span><span class="speak-word-inline" data-audio-us-female="/audios/hot-us-female.mp3" data-audio-us-male="/audios/hot-us-male.mp3"></span>。所以，按照《劍橋英語發聲詞典》的標註，我們可以這麼理解：在美式英語中，<span class="pho">ʌ</span> 最短，<span class="pho">ɑː</span> 更長一些，而 <span class="pho">ɑːr</span> 最長。

## 3.1.1.4. 兒化音 <span class="pho">ɑːr</span>

<span class="pho">ɑːr</span> 的兒化音得用**舌背後縮**的方式發出（*Bunched R*）。如果你肯仔細體會的話，從感覺上來看，<span class="pho">ɑːr</span> 更像是 <span class="pho">ɑːᵊr</span>，或者寫成 <span class="pho">ɑːɚ</span> —— 會有個實際上相當明顯的 <span class="pho">ᵊ</span> 的加音，比如 *card* <span class="pho alt">kɑːrd</span> <span class="speak-word-inline" data-audio-us-male="/audios/card-us-male.mp3" data-audio-us-female="/audios/card-us-female.mp3"></span>。

## 3.1.1.5. <span class="pho">ɑː</span> ⭢ <span class="pho">æ</span>

除了《劍橋英語發聲詞典》裡的 <span class="pho">ɑː</span> 還對應著英式英語中的短母音 <span class="pho">ɒ</span> 之外，另外一個值得注意的是，英音中的 <span class="pho">ɑː</span> 在美式英語中讀作 <span class="pho">æ</span>，最常見的例子是 *ask*，英音是 <span class="pho alt">ɑːsk</span><span class="speak-word-inline" data-audio-uk-male="/audios/ask-uk-male.mp3" data-audio-uk-female="/audios/ask-uk-female.mp3"></span>，美音是 <span class="pho alt">æsk</span><span class="speak-word-inline" data-audio-us-male="/audios/ask-us-male.mp3" data-audio-us-female="/audios/ask-us-female.mp3"></span>。
