## 3.2.12.3. 加音 <span class="pho">ᵊ</span>

<span class="pho">l</span> 的困難總是來自於它們處在音節末尾的時候。

不妨嘗試著先讀一下 *milk* <span class="pho alt">mɪlk</span><span class="speak-word-inline" data-audio-us-male="/audios/milk-us-male.mp3" data-audio-us-female="/audios/milk-us-female.mp3"></span>…… 很多人讀出來的實際上是 <span class="pho alt">mjʊk</span>…… 另外一個發音類似的常用單詞也常常被誤讀，*build* <span class="pho alt">bɪld</span> <span class="speak-word-inline" data-audio-us-male="/audios/build-us-male.mp3" data-audio-us-female="/audios/build-us-female.mp3"></span>，基本上總是被誤讀為 <span class="pho alt">bjʊld</span>。

<span class="pho">l</span> 在音節末尾的時候，有一個看不到的動作，舌尖會向前向上移動直至貼到齦脊。於是，音節末尾的 <span class="pho">l</span> 之前會有一個加音，輕微的 <span class="pho">ᵊ</span>。

說實話，倒也不是特別難，這只不過是幾分鐘就可以解決的問題。

> * 先反覆說 <span class="pho">ə... əl... ə... əl...</span> 這裡格外需要注意：嘴唇張開之後沒有其它動作，**不要帶嘴唇變化造成的 <span class="pho">ʊ</span> 的音色**；從 <span class="pho">ə</span> 到 <span class="pho">l</span> 的過程中，保持聲帶振動，剩餘的全靠舌尖移動完成……
> * 接著直接反覆說 <span class="pho">əl... əl...</span>
> * 再反覆說 <span class="pho">əlk... əlk...</span>
> * 現在反覆說 <span class="pho">mɪᵊlk... mɪᵊlk...</span> 

> [!Note]
> 需要格外注意的有兩點：
> * <span class="pho">ɪ</span> 不是 <span class="pho">iː</span> 的簡單化短版本（[3.1.3.2](3.1.3-i#_3-1-3-2-短母音-ɪ)），它更接近 <span class="pho">eɪ</span> 的短版本。於是，milk 是 <span class="pho alt">mɪlk</span>，不是 <span class="pho alt">milk</span>，你哪怕把它讀成 <span class="pho alt">meɪlk</span> 都可能更準確一些 —— 只不過得想辦法把 <span class="pho">eɪ</span> 讀得短一點<span class="speak-word-inline" data-audio-us-male="/audios/milk-us-male.mp3" data-audio-us-female="/audios/milk-us-female.mp3"></span>……
> * <span class="pho">l</span> 這個音，在音節末尾的時候，舌尖有個向前移動至齦脊（位置 ③）的動作 —— 這個動作很重要，否則與它相關的連讀就說不出來（[4.4.1](4.4-linking#_4-4-1-子音-母音-consonant-vowel)）。很多人會用 <span class="pho">ʊ</span> 替代這個動作，比如，吧 *result* <span class="pho alt">rɪˈzəlt</span> 讀成 <span class="pho alt">rɪˈzəʊt</span>；同樣，<span class="pho alt">mɪlk</span> 也實際上被很多人讀作 <span class="pho alt">mjʊk</span>…… 必須注意，音節末尾的 <span class="pho">l</span> 只有舌尖動作，**沒有嘴唇動作**。再用同樣的方式練練 *girl* <span class="pho alt">ɡɝːl</span><span class="speak-word-inline" data-audio-us-male="/audios/girl-us-male.mp3" data-audio-us-female="/audios/girl-us-female.mp3"></span>（常常會被誤讀成 <span class="pho alt">gəʊ</span>）, *world* <span class="pho alt">wɝːld</span><span class="speak-word-inline" data-audio-us-male="/audios/world-us-male.mp3" data-audio-us-female="/audios/world-us-female.mp3"></span>（常常會被誤讀成 <span class="pho alt">wəʊd</span>）。

以 <span class="pho">l</span> 處於音節末尾的常用詞彙很多，讀得時候要注意，要加上 <span class="pho">ᵊ</span>，比如：

> * *fail* <span class="pho alt">feɪl</span> ⭢ <span class="pho alt">feɪᵊl</span><span class="speak-word-inline" data-audio-us-male="/audios/fail-us-male.mp3" data-audio-us-female="/audios/fail-us-female.mp3"></span>
> * *file* <span class="pho alt">faɪl</span> ⭢ <span class="pho alt">faɪᵊl</span><span class="speak-word-inline" data-audio-us-male="/audios/file-us-male.mp3" data-audio-us-female="/audios/file-us-female.mp3"></span>
> * *sell* <span class="pho alt">sel</span> ⭢ <span class="pho alt">seᵊl</span><span class="speak-word-inline" data-audio-us-male="/audios/sell-us-male.mp3" data-audio-us-female="/audios/sell-us-female.mp3"></span>
> * *sale* <span class="pho alt">seɪl</span> ⭢ <span class="pho alt">seɪᵊl</span><span class="speak-word-inline" data-audio-us-male="/audios/sale-us-male.mp3" data-audio-us-female="/audios/sale-us-female.mp3"></span>
> * *tell* <span class="pho alt">tel</span> ⭢ <span class="pho alt">teᵊl</span><span class="speak-word-inline" data-audio-us-male="/audios/tell-us-male.mp3" data-audio-us-female="/audios/tell-us-female.mp3"></span>
> * *tale* <span class="pho alt">teɪl</span> ⭢ <span class="pho alt">teɪᵊl</span><span class="speak-word-inline" data-audio-us-male="/audios/tale-us-male.mp3" data-audio-us-female="/audios/tale-us-female.mp3"></span>
> * *trail* <span class="pho alt">treɪl</span> ⭢ <span class="pho alt">treɪᵊl</span><span class="speak-word-inline" data-audio-us-male="/audios/trail-us-male.mp3" data-audio-us-female="/audios/trail-us-female.mp3"></span>
> * *drill* <span class="pho alt">drɪl</span> ⭢ <span class="pho alt">drɪᵊl</span><span class="speak-word-inline" data-audio-us-male="/audios/drill-us-male.mp3" data-audio-us-female="/audios/drill-us-female.mp3"></span>

> [!Note]
> 
> <span class="pho alt">ʌl</span> 沒有 <span class="pho">ᵊ</span> 的加音，因為 <span class="pho">ʌ</span> 和 <span class="pho">ə</span> 原本就非常相近。比如，*ultimate* <span class="pho alt">ˈʌltɪmət</span><span class="speak-word-inline" data-audio-us-male="/audios/ultimate-us-male.mp3" data-audio-us-female="/audios/ultimate-us-female.mp3"></span>，或者 *result* <span class="pho alt">rɪˈzəlt</span><span class="speak-word-inline" data-audio-us-male="/audios/result-us-male.mp3" data-audio-us-female="/audios/result-us-female.mp3"></span>。另外，在自然語流中，<span class="pho alt">iːl</span> 要注意。比如， *reveal*，詞典裡的音標是 <span class="pho alt">rɪˈviːl</span>， 但它的確讀做 <span class="pho alt">rɪˈviːᵊl</span><span class="speak-word-inline" data-audio-us-male="/audios/reveal-us-male.mp3" data-audio-us-female="/audios/reveal-us-female.mp3"></span>，有加音 <span class="pho">ᵊ</span>，但，*revealing*，讀作 <span class="pho alt">rɪˈviːlɪŋ</span><span class="speak-word-inline" data-audio-us-male="/audios/revealing-us-male.mp3" data-audio-us-female="/audios/revealing-us-female.mp3"></span>，沒有加音 <span class="pho">ᵊ</span>，可能是因為 <span class="pho">l</span> 被挪到後面和 <span class="pho">ɪŋ</span> 構成了第三個音節 <span class="pho alt">lɪŋ</span>……

<span class="pho">l</span> 在語言學裡有個專門的稱呼，叫做**音節子音**（<span class="pho">l</span>, <span class="pho">r</span>, <span class="pho">m</span>, <span class="pho">n</span>, 都是音節子音），就是因為它的發聲動作如果完整的話，前面就會自然附加著一個輕微的母音 <span class="pho">ᵊ</span> 的音色。比如，*little* <span class="pho alt">ˈlɪtl̩</span><span class="speak-word-inline" data-audio-us-male="/audios/little-us-male.mp3" data-audio-us-female="/audios/little-us-female.mp3"></span> —— <span class="pho">l</span> 下面多出的那一個點 <span class="pho">l̩</span> 的意思是說，這裡的 <span class="pho">l̩</span> 是個**音節子音**，相當於 <span class="pho alt">ᵊl</span>…… 當然，很多詞典乾脆就直接標註成了 <span class="pho alt">ˈlɪtᵊl</span> 或者 <span class="pho alt">ˈlɪtəl</span>。《劍橋英語發聲詞典》中沒有音節子音符號，也沒區分 <span class="pho">ᵊ</span> 和 <span class="pho">ə</span>，標註的是 <span class="pho alt">ˈlɪt̬əl</span>（另外要注意：因為 <span class="pho">t</span> 夾在兩個母音之間，變成了彈舌音 <span class="pho">t̬</span>）。

## 3.2.12.4. 舌尖起始位置比較

比較一下以下的子音發聲時**舌尖起始位置**的不同：

![articulator-tongue-tip-positions](/images/articulator-tongue-tip-positions.svg)

> * <span class="pho">t/d</span> 的舌尖起始位置時貼在齦脊的前部 ③；
> * <span class="pho">s/z</span> 要相對於 <span class="pho">t/d</span> 往前一些，貼近但沒有接觸到牙齒和牙齦的接縫處 ②；
> * <span class="pho">θ/ð</span> 要相對於 <span class="pho">s/z</span>再往前一些，貼在牙齒上，或者伸出牙齒一點點 ①；
> * <span class="pho">ʃ/ʒ</span> 和 <span class="pho">t/d</span> 一樣，接近但沒有接觸到齦脊的那個部位 ③；
> * <span class="pho">tʃ/dʒ</span> 和 <span class="pho">t/d</span> 一樣，貼在齦脊的那個部位 ③；
> * <span class="pho">l</span> 和 <span class="pho">t/d</span> 一樣，貼在齦脊的那個部位 ③；
> * <span class="pho">r</span> 相對於 <span class="pho">ʃ/ʒ</span> 還要再往後一點，貼在齦脊的那個部位 ④……

| Positions of tongue tip | Consonants                                                                                                                                                                                                                                                     |
| :---------------------: | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
|            ①            | <span class="pho">θ</span>, <span class="pho">ð</span>                                                                                                                                                                                                         |
|            ②            | <span class="pho">s</span>, <span class="pho">z</span>                                                                                                                                                                                                         |
|            ③            | <span class="pho">t</span>, <span class="pho">d</span>, <span class="pho">l</span>, <span class="pho">ʃ</span>, <span class="pho">ʒ</span>, <span class="pho">tʃ</span>, <span class="pho">dʒ</span>, <span class="pho">tr</span>, <span class="pho">dr</span> |
|            ④            | <span class="pho">r</span>                                                                                                                                                                                                                                     |
