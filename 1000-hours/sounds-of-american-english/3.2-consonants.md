# 3.2. 子音（Consonants）

地球上的所有語言都一樣，在自然語流中，**子音**不會單獨存在。單獨的**母音**，可以構成**音節**，也可以構成或者表達意義。比如，*ah* <span class="pho alt">ɑː</span><span class="speak-word-inline" data-audio-us-male="/audios/us_phonetics_sound_father_2023feb.mp3"></span>, *oh* <span class="pho alt">oʊ</span><span class="speak-word-inline" data-audio-us-male="/audios/us_phonetics_sound_nose_2023feb.mp3"></span>，或者 *Uh-Oh* <span class="pho alt">ˈʌ.oʊ</span><span class="speak-word-inline" data-audio-us-female="/audios/Uh-Oh.mp3"></span>[^1]。但，絕大多數**子音**必須依附於**母音**才能構成**音節**，並且，單獨的子音也不能構成意義。

子音可以根據發聲時**聲帶**是否震動被分為清子音（voiceless consonants）與濁子音（voiced consonants）。清子音只有氣流震動，沒有聲帶振動，而濁子音的氣流震動卻伴有聲帶振動。比如，<span class="pho">s</span> 是清子音，<span class="pho">z</span> 是濁子音。

|                     | 子音（consonants）                                                                                                                                                                                                                                                                                                                                                                                                                    |
| ------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 清子音（voiceless） | <span class="pho">p</span>, <span class="pho">t</span>, <span class="pho">k</span>, <span class="pho">f</span>, <span class="pho">s</span>, <span class="pho">θ</span>, <span class="pho">ʃ</span>, <span class="pho">h</span>, <span class="pho">tʃ</span>, <span class="pho">tr</span>, <span class="pho">ts</span>                                                                                                                 |
| 濁子音（voiced）    | <span class="pho">b</span>, <span class="pho">d</span>, <span class="pho">g</span>, <span class="pho">v</span>, <span class="pho">z</span>, <span class="pho">ð</span>, <span class="pho">ʒ</span>, <span class="pho">r</span>, <span class="pho">dʒ</span>, <span class="pho">dr</span>, <span class="pho">dz</span>, <span class="pho">m</span>, <span class="pho">n</span>, <span class="pho">ŋ</span>, <span class="pho">l</span> |

注意：<span class="pho">w</span> 和 <span class="pho">j</span> 這兩個子音被稱為**半母音** —— 無法用聲帶振動與否分類。

子音的另外一個劃分，是根據口腔內**氣流**振動的方式和位置。

| 氣流振動方式（Airflow） | 清濁與否（Voiced or not） | 子音（Consonants）                                                                                                                         |
| ----------------------- | ------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------ |
| 破擦音（Plosive）       | 清子音                    | <span class="pho">p</span>, <span class="pho">t</span>, <span class="pho">k</span>                                                         |
|                         | 濁子音                    | <span class="pho">b</span>, <span class="pho">d</span>, <span class="pho">g</span>                                                         |
| 摩擦音（Fricative）     | 清子音                    | <span class="pho">f</span>, <span class="pho">s</span>, <span class="pho">θ</span>, <span class="pho">ʃ</span>, <span class="pho">h</span> |
|                         | 濁子音                    | <span class="pho">v</span>, <span class="pho">z</span>, <span class="pho">ð</span>, <span class="pho">ʒ</span>                             |
| 塞擦音（Affricate）     | 清子音                    | <span class="pho">tʃ</span>, <span class="pho">tr</span>, <span class="pho">ts</span>                                                      |
|                         | 濁子音                    | <span class="pho">dʒ</span>, <span class="pho">dr</span>, <span class="pho">dz</span>                                                      |
| 鼻音（Nasal）           | 濁子音                    | <span class="pho">m</span>, <span class="pho">n</span>, <span class="pho">ŋ</span>                                                         |
| 邊擦音（Lateral）       | 濁子音                    | <span class="pho">l</span>, <span class="pho">r</span>                                                                                     |
| 半母音（Semivowel）     |                           | <span class="pho">j</span>, <span class="pho">w</span>                                                                                     |

作為初學者，**你無需嘗試記憶這些分類和術語** —— 除非將來你有多餘的時間精力。這些語言學的概念，並不構成任何學習障礙。你需要了解的只不過是分類的**依據**：**聲帶振動與否**，以及**氣流振動方式和位置**。

另外，需要注意：有 *4* 個子音，<span class="pho">l</span>, <span class="pho">m</span>, <span class="pho">n</span>，以及之前見過很多次的 <span class="pho">r</span>，它們被稱作 “**音節子音**”，因為它們自帶 <span class="pho">ə</span> 的音色。比如，*bottle* <span class="pho alt">ˈbɑːtl</span>, *bottom* <span class="pho alt">ˈbɑːtm</span>, *button* <span class="pho alt">ˈbʌtn</span>。有些詞典會用 <span class="pho">ᵊ</span>，比如寫成：
*bottle* <span class="pho alt">ˈbɑːtᵊl</span>, *bottom* <span class="pho alt">ˈbɑːtᵊm</span>, *button* <span class="pho alt">ˈbʌtᵊn</span>。而《劍橋英語發聲詞典》的標註方式是：*bottle* <span class="pho alt">ˈbɑːt̬əl</span>, *bottom* <span class="pho alt">ˈbɑːt̬əm</span>, *button* <span class="pho alt">ˈbʌtən</span>……

格外需要注意的是：子音發聲時，**舌尖起始位置**非常重要。
