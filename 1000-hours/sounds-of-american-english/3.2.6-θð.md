# 3.2.6. <span class="pho">θ/ð</span>

對很多人來說，<span class="pho">θ/ð</span> 相對較難 —— 主要是因為很多語言中沒有這兩個音，比如，中日韓都一樣，都沒有這兩個音。

雖然，學校裡的英語老師們都強調說，發這兩個音的時候要用牙齒咬住一點點舌尖再發聲，也就是說，舌尖要伸出牙齒一點點 —— 這麼做當然發聲更為清楚，在讀單個音節或詞彙的時候尤為明顯……

但是，在**連貫的自然語流**之中，會有大量處於**非重音**位置的 <span class="pho">θ/ð</span> 發聲時舌尖起始位置並沒有伸到牙齒之外，只是**貼在牙齒**上而已 —— 即，位置 ①。反過來，即便是處於**重音**位置的 <span class="pho">θ/ð</span>, 也一樣可以把發聲動作**簡化**一下，只要把舌尖貼到牙齒上，而後開始發音即可。

最初練習的時候，可以藉助 <span class="pho">s/z</span> 作為輔助:

> * 先發 <span class="pho">s</span> 的音，而後，把舌尖往前推一點點就接觸到了牙齒，然後以與 <span class="pho">s</span> 同樣的方式發音，就能聽到 <span class="pho">θ</span>；然後分別說兩個詞 *sink* <span class="pho alt">sɪŋk</span><span class="speak-word-inline" data-audio-us-male="/audios/sink-us-male.mp3" data-audio-us-female="/audios/sink-us-female.mp3"></span> 和 *think* <span class="pho alt">θɪŋk</span><span class="speak-word-inline" data-audio-us-male="/audios/think-us-male.mp3" data-audio-us-female="/audios/think-us-female.mp3"></span>……
> * 先發 <span class="pho">z</span> 的音，而後，把舌尖往前推一點點就接觸到了牙齒，然後以與 <span class="pho">z</span> 同樣的方式發音，就能聽到 <span class="pho">ð</span>；然後分別說兩個詞 *zed* <span class="pho alt">zed</span><span class="speak-word-inline" data-audio-us-male="/audios/zed-us-male.mp3" data-audio-us-female="/audios/zed-us-female.mp3"></span> 和 *that* <span class="pho alt">ðæt</span><span class="speak-word-inline" data-audio-us-male="/audios/that-us-male.mp3" data-audio-us-female="/audios/that-us-female.mp3"></span>……
