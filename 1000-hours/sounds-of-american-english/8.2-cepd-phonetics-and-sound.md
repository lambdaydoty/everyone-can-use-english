# 8.2. 獲取 CEPD 音標

在 Github 上有一個開源的倉庫，提供了《劍橋英語發聲詞典》的 json 格式資料庫：

> https://github.com/zelic91/camdict

將這個倉庫裡的 [cam_dict.refined.json](https://github.com/zelic91/camdict/raw/main/cam_dict.refined.json) 下載並儲存到本地某個位置。

我寫了一個 Alfred 的 workflow，使用的是 macOS 系統自帶的 *python3*：`/usr/bin/python3`，透過設定關鍵字啟動一個 Python 指令碼，查詢某個單詞（甚至整個句子）在《劍橋英語發聲詞典》（CEPD）中的音標。

> [CEPD-phonetic-transcription.alfredworkflow](https://1000h.org/alfred-workflows/CEPD-phonetic-transcription.alfredworkflow)

下載這個檔案之後，匯入 Alfred。

在使用之前要注意：

> * 修改各個 Python 指令碼內的 `cam_dict.refined.json` 的檔案路徑。

這個 workflow 可用的啟動關鍵字分別是：

> * `cams`：查詢音標（美式發音）
> * `camk`：查詢音標（英式發音）
> * `camsd`：用瀏覽器開啟 CEPD 真人示範錄音（美式發音）線上網址
> * `camkd`：用瀏覽器開啟 CEPD 真人示範錄音（英式發音）線上網址
> * `camw`：用瀏覽器開啟 CEPD 查詢頁面
> * `ipa`：返回 CMU（卡耐基梅隆大學）音標庫中的音標

以下是查詢音標的 workflow（啟動關鍵字為 `cams`）中的 python 指令碼：

```python
#
# NOTE: Python 2 is deprecated in macOS, and has been removed from macOS 12.3+
#
import sys
import json

# 假設你的 JSON 資料庫是一個 JSON 檔案，我們將從檔案中載入資料
# 如果 JSON 資料在記憶體中或其他格式，你可能需要修改這部分程式碼
def load_json_database(file_path):
    records = []
    with open(file_path, 'r') as file:
        for line in file:
            try:
                record = json.loads(line)
                records.append(record)
            except json.JSONDecodeError as e:
                print(f"Error parsing JSON: {e}")
    return records

# 在 JSON 資料庫中檢索 word
def search_in_json_database(database, search_word, region):
    for record in database:
        # 檢查 word 欄位是否匹配
        if record.get('word') == search_word:
            # 找到匹配項後，獲取美式發音資訊
            pos_items = record.get('pos_items', [])
            for pos_item in pos_items:
                pronunciations = pos_item.get('pronunciations', [])
                for pronunciation in pronunciations:
                    if pronunciation.get('region') == region:
                        # 找到美式發音，返回相關資訊
                        return {
                            'pronunciation': pronunciation.get('pronunciation'),
                            'audio': pronunciation.get('audio')
                        }
    # 如果沒有找到匹配的 word 欄位，返回 'not exist'
    return 'not exist'

# cam_dict.refined.json 的檔案路徑
json_db_file_path = '/Users/joker/github/camdict/cam_dict.refined.json'

# 要檢索的單詞
search_word = sys.argv[1]

region = "us"

json_database = load_json_database(json_db_file_path)

# replace punctuations in text with space
punctuations = ",.?!;"
for p in punctuations:
    search_word = search_word.replace(p, " ")
words = [word for word in search_word.split() if word.strip() != '']

phonetics = []

for w in words:
  # 檢索並獲取結果
  w = w.strip().lower()

  if w[-1] in punctuations:
    w = w.rstrip(",.?!;")
  result = search_in_json_database(json_database, w, region)

  if result == 'not exist':
    phonetics.append(w+"*")
  else:
    phonetics.append(result['pronunciation'])

returnvalue = ''
for p in phonetics:
  returnvalue += p + ' '

sys.stdout.write(returnvalue.strip())
```
