# 3.2.2. <span class="pho">t</span>/<span class="pho">d</span>

<span class="pho">t</span> 是清子音，<span class="pho">d</span> 是濁子音。<span class="pho">d</span> 相對簡單；然而，<span class="pho">t</span> 卻可能是美式英語裡最複雜的音素，所以也是最需要專門練習的子音之一。

## 3.2.2.2. 舌尖起始位置

雖然每種語言都有這兩個子音，但，英文的 <span class="pho">t/d</span> 的發聲關鍵在於**舌尖起始位置** —— 頂在**齦脊**（*alveolar ridge*）上，即，位置 ③，而不是貼在牙齒上。

在中文普通話裡，<span class="pho">t/d</span> 這兩個聲母發聲時舌尖起始位置貼在牙齒上；你可以試著用中文普通話說一遍 “我踢死你！”（<span class="pho">wǒ tī sǐ nǐ</span>）<span class="speak-word-inline" data-audio-other="/audios/我踢死你-zh-cn-male.mp3"></span>……

所以，對中國人來說，這其實是個相當難以糾正的習慣，因為舌尖總是不由自主地使用母語習慣貼到牙齒上（位置 ①）而不是齦脊上（位置 ③） —— 於是，只能透過大量且高密度的練習建立新習慣。一個很經典的練習詞彙是 *student* <span class="pho alt">ˈstuː.dənt</span><span class="speak-word-inline" data-audio-us-male="/audios/student-us-male.mp3" data-audio-us-female="/audios/student-us-female.mp3"></span>。

## 3.2.2.3. 並非 <span class="pho">tə/də</span>

主要額外提醒的是，雖然《劍橋英語發聲詞典》中，<span class="pho">t/d</span> 的示例都讀成了 <span class="pho">tə</span><span class="speak-word-inline" data-audio-us-male="/audios/us_phonetics_sound_town_2023feb.mp3"></span> 和 <span class="pho">də</span><span class="speak-word-inline" data-audio-us-male="/audios/us_phonetics_sound_day_2023feb_001.mp3"></span>，但那並不是這兩個音素在自然語流中的實際發音。

就好像 <span class="pho">p/b</span> 一樣，<span class="pho">t/d</span> 在實際發音的時候也會自然帶著一個很輕的 <span class="pho">ɤ</span>，而不是 <span class="pho">ə</span>。若是依據母語的習慣，中國人習慣於將 <span class="pho">t(ɤ)</span> 讀作 <span class="pho">tə</span>，將 <span class="pho">d(ɤ)</span> 讀作 <span class="pho">də</span>。還是需要為此專門建立一個新習慣。

## 3.2.2.3. <span class="pho">t</span> 的變體

<span class="pho">t</span> 這個音素 —— 當然也是最常用的一個 —— 也許是美式英語的子音裡最複雜的一個。t 只有在處於重音音節開頭的時候才會被正常讀出，比如，*take* <span class="pho alt">teɪk</span><span class="speak-word-inline" data-audio-us-male="/audios/take-us-male.mp3" data-audio-us-female="/audios/take-us-female.mp3"></span>。

除了正常音 <span class="pho">t</span> 之外，它有很多變體（*allophone*），比如，彈舌音 <span class="pho">t̬</span>，阻塞音 <span class="pho">t̚</span>，省略音/可選音 <span class="pho">ᵗ</span>，以及很多人不經講解就完全注意不到的喉塞音 <span class="pho">ʔ</span>。

### 3.2.2.3.1. 彈舌音 <span class="pho">t̬</span>

夾在兩個母音中間的 <span class="pho">t</span> 會變成接近 <span class="pho">d</span> 的彈舌音 <span class="pho">t̬</span>，比如，*water* <span class="pho alt">ˈwɑːt̬ɚ</span><span class="speak-word-inline" data-audio-us-male="/audios/water-us-male.mp3" data-audio-us-female="/audios/water-us-female.mp3"></span>。《劍橋英語發聲詞典》中的美式音標專門為此做了最佳化，增加了彈舌音 <span class="pho">t̬</span> 的標註 —— 大多數詞典並非如此。

### 3.2.2.3.2. 阻塞音 <span class="pho">t̚</span>

處於音節末尾的 <span class="pho">t</span>，有時只是舌尖動作完整，卻沒有發出聲音。即，舌尖頂到齦脊之後並沒有送出氣流。比如，*but* <span class="pho alt">bʌt̚</span><span class="speak-word-inline" data-audio-us-male="/audios/but-stop-us-male.mp3" data-audio-us-female="/audios/but-stop-us-female.mp3"></span>。

### 3.2.2.3.3. 省略音/可選音 <span class="pho">ᵗ</span>

處於音節末尾 <span class="pho">nt</span>，其中的 <span class="pho">t</span> 可能會被省略掉，比如，*winter* <span class="pho alt">ˈwɪnᵗɚ</span><span class="speak-word-inline" data-audio-us-male="/audios/winter-optional-us-male.mp3" data-audio-us-female="/audios/winter-optional-us-female.mp3"></span> —— 當然，若是讀成 <span class="pho alt">ˈwɪntɚ</span><span class="speak-word-inline" data-audio-us-male="/audios/winter-us-male.mp3" data-audio-us-female="/audios/winter-us-female.mp3"></span>也沒錯……[^1] 。

### 3.2.2.3.4. 喉塞音 <span class="pho">ʔ</span>

喉塞音 <span class="pho">ʔ</span> 又是一個英語音標體系中都沒有收錄的國際音標符號。

喉塞音 <span class="pho">ʔ</span> 相當於是 <span class="pho">g</span> 這個音的阻塞音 <span class="pho">g̚</span>, 就好像用阻塞音說 big 時一樣，舌背後部抬起來頂住軟顎，而後並不離開，所以沒有氣流輸出，感覺聲音被卡住了一小下…… 再反覆聽聽 *Uh-Oh!* <span class="pho alt">ʌʔoʊ</span><span class="speak-word-inline" data-audio-us-female="/audios/Uh-Oh.mp3"></span> 就明白了。

<span class="pho">m/n/l</span> 之前的 <span class="pho">t</span> 有可能變成喉塞音 <span class="pho">ʔ</span>，比如，*utmost* <span class="pho alt">ˈʌʔˌmoʊst</span><span class="speak-word-inline" data-audio-us-male="/audios/utmost-us-male.mp3" data-audio-us-female="/audios/utmost-us-female.mp3"></span>, *fitness* <span class="pho alt">ˈfɪʔnəs</span><span class="speak-word-inline" data-audio-us-male="/audios/fitness-us-male.mp3" data-audio-us-female="/audios/fitness-us-female.mp3"></span>, *witness* <span class="pho alt">ˈwɪʔnəs</span><span class="speak-word-inline" data-audio-us-male="/audios/witness-us-male.mp3" data-audio-us-female="/audios/witness-us-female.mp3"></span>, 或 *fruitless* <span class="pho alt">ˈfruːʔləs</span><span class="speak-word-inline" data-audio-us-male="/audios/fruitless-us-male.mp3" data-audio-us-female="/audios/fruitless-us-female.mp3"></span> —— 當然，所有詞典都一樣，都統一標註成 <span class="pho">t</span> 而不是 <span class="pho">ʔ</span>。

### 3.2.2.3.5. <span class="pho">t</span> 的濁化

<span class="pho">t</span> 處於 <span class="pho">s</span> 之後的時候會**濁化**，會被讀成 <span class="pho">d</span> —— 雖然詞典裡的音標標註還是 <span class="pho">t</span>，比如 *student*，字典裡的標註依然是 <span class="pho alt">ˈstuː.dənt</span>，但讀出來的卻是 <span class="pho alt">ˈsduː.dənt</span><span class="speak-word-inline" data-audio-us-male="/audios/student-us-male.mp3" data-audio-us-female="/audios/student-us-female.mp3"></span>
。

## 3.2.2.4. <span class="pho">d</span> 的清化

<span class="pho">t</span> 有可能**濁化**…… 與之相對地，<span class="pho">d</span> 也可能**清化**。比如，*ed* 有時並不讀作 <span class="pho alt">ɪd</span>, 在清子音 <span class="pho">p</span>, <span class="pho">k</span>, <span class="pho">f</span>, <span class="pho">s</span>, <span class="pho">ʃ</span>, <span class="pho">tʃ</span>, <span class="pho">θ</span> 之後的時候，讀作 <span class="pho alt">t</span>。比如，*hoped* <span class="pho alt">hoʊpt</span> <span class="speak-word-inline" data-audio-us-male="/audios/hoped-us-male.mp3" data-audio-us-female="/audios/hoped-us-female.mp3"></span>，*laughed* <span class="pho alt">læft</span> <span class="speak-word-inline" data-audio-us-male="/audios/laughed-us-male.mp3" data-audio-us-female="/audios/laughed-us-female.mp3"></span>，*liked* <span class="pho alt">laɪkt</span> <span class="speak-word-inline" data-audio-us-male="/audios/liked-us-male.mp3" data-audio-us-female="/audios/liked-us-female.mp3"></span>，*watched* <span class="pho alt">wɑːtʃt</span> <span class="speak-word-inline" data-audio-us-male="/audios/watched-us-male.mp3" data-audio-us-female="/audios/watched-us-female.mp3"></span>。
