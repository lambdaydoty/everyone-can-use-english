雖然發聲時舌尖起始位置相同，但，<span class="pho">tr</span>/<span class="pho">dr</span> 與 <span class="pho">tʃ</span>/<span class="pho">dʒ</span> 不一樣的地方在於，<span class="pho">tr</span>/<span class="pho">dr</span> 在發聲之前嘴唇要有個縮小成圓形的動作，所以它們開頭的音節，總是聽起來夾帶著一個 <span class="pho">ʷ</span>，*tree* <span class="pho alt">triː</span> 聽起來是 <span class="pho alt">tʃʷiː</span><span class="speak-word-inline" data-audio-us-male="/audios/tree-us-male.mp3" data-audio-us-female="/audios/tree-us-female.mp3"></span>，*dry* <span class="pho alt">draɪ</span> 聽起來是 <span class="pho alt">dʒʷaɪ</span><span class="speak-word-inline" data-audio-us-male="/audios/dry-us-male.mp3" data-audio-us-female="/audios/dry-us-female.mp3"></span>。

很多地區的人基於自身母語的習慣，會把 <span class="pho">tr</span> 和 <span class="pho">dr</span> 分開著連續讀出的。比如，*tree*，印度人可能會把它讀成 <span class="pho alt">tɤ.riː</span><span class="speak-word-inline" data-audio-other="/audios/tree-ru-ru.mp3"></span>，其中的 <span class="pho">t</span> 乾脆讀成 <span class="pho">d</span>，並且 <span class="pho">r</span> 還會用齒齦顫音；再比如，*driver*，可能會被俄羅斯人讀成 <span class="pho alt">dɤ.raɪvə</span><span class="speak-word-inline" data-audio-other="/audios/driver-ru-ru.mp3"></span>，其中的 <span class="pho">r</span> 同樣會用齒齦顫音。

其實，他們只要把 <span class="pho">tr</span> 理解為 <span class="pho">tʃ</span> + <span class="pho">r</span>，把 <span class="pho">dr</span> 理解為 <span class="pho">dʒ</span> + <span class="pho">r</span>，就好辦了，挺簡單的。
