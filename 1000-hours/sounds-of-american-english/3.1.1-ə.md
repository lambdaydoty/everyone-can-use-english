# 3.1.1. <span class="pho">ə</span>/<span class="pho">ɚ</span>/<span class="pho">ɝː</span>

<span class="pho">ə</span> 這個基礎母音，在英語中最為普遍。在美式英語中，也相對特殊，因為它的長短兩個版本都常常帶著兒化音，所以值得第一個詳細講解。

在 6 個基礎母音中，<span class="pho">ə</span> 和 <span class="pho">i</span> 的口型一樣且最小。

## 3.1.1.2. 央母音 <span class="pho">ə</span>

在美式英語中，<span class="pho">ə</span> 有個專門的名稱，叫做 “央母音”（*schwa*）。

> 在連貫的自然語流中，很多非重音音節的母音，都有被簡化成 <span class="pho">ə</span> 的傾向，尤其是 <span class="pho">ɪ</span>。比如，*individual*, <span class="pho alt">ˌɪndəˈvɪdʒuəl</span>，實際上會被讀作 <span class="pho alt">ˌɪndəˈvɪdʒəʷəl</span><span class="speak-word-inline" data-audio-us-male="/audios/individual-us-male.mp3" data-audio-us-female="/audios/individual-us-female.mp3"></span> —— <span class="pho">u</span> 也變成了 <span class="pho">ə</span>，而後 <span class="pho">u</span> 和 <span class="pho">ə</span> 之間有個加音 <span class="pho">ʷ</span>…… 事實上，現在已經有很多詞典乾脆直接標註成了後者。

在非重音音節上的 <span class="pho">ə</span> 是不可能帶有兒化音的 —— 這一點要格外注意。很多人會矯枉過正，任何時候都把 <span class="pho">ə</span> 讀成 <span class="pho">ɚ</span>…… 比如，有些人可能會把 *focus* <span class="pho alt">ˈfoʊkəs</span><span class="speak-word-inline" data-audio-us-male="/audios/focus-us-male.mp3" data-audio-us-female="/audios/focus-us-female.mp3"></span> 錯讀成 <span class="pho alt">ˈfoʊkɚs</span><span class="speak-word-inline" data-audio-other="/audios/focurs-us.mp3"></span>。

## 3.1.1.3. 兒化音

短母音 <span class="pho">ɚ</span> 和長母音 <span class="pho">ɝː</span> 這兩個兒化音，有兩種發聲方式，一種是憑直覺捲起舌尖（無論是前卷還是後卷，都被稱作 *Retroflex R*），這也是絕大多數初學者發這個音的唯一選擇。

由於這個音發聲時的舌頭動作是別人看不見的，所以，人們在學這個發音的時候，無法猜到第二種方式（被稱作 *Bunched R*）：舌背部分抬起來，略微後縮，使舌的兩側接觸到後牙。舌背的中心部分較低，空氣透過這個凹槽產生聲音。

![articulator-tongue-tip-ɚ](/images/articulator-tongue-tip-ɚ.svg)

以下是另外一個角度的圖示[^1]：

![retroflex-bunched-r](/images/retroflex-bunched-r.svg)

以下的影片更為直觀，*worker* <span class="pho alt">ˈwɝː.kɚ</span><span class="pho alt">ˈwɜː.kə</span><span class="speak-word-inline" data-audio-us-female="/audios/worker-us-female.mp3" data-audio-us-male="/audios/worker-us-male.mp3"></span> 這個詞的長短兩個版本的兒化音都是用 *Bunched R* 讀出來的[^2]：

事實上，其他的兒化音也都可以用這種方式發聲。不妨用這種方式重新試試 *card* <span class="pho alt">kɑːrd</span><span class="speak-word-inline" data-audio-us-male="/audios/card-us-male.mp3" data-audio-us-female="/audios/card-us-female.mp3"></span>, *mark* <span class="pho alt">mɑːrk</span><span class="speak-word-inline" data-audio-us-male="/audios/mark-us-male.mp3" data-audio-us-female="/audios/mark-us-female.mp3"></span>；再試試 *door* <span class="pho alt">dɔːr</span><span class="speak-word-inline" data-audio-us-male="/audios/door-us-male.mp3" data-audio-us-female="/audios/door-us-female.mp3"></span>, *short* <span class="pho alt">ʃɔːrt</span><span class="speak-word-inline" data-audio-us-male="/audios/short-us-male.mp3" data-audio-us-female="/audios/short-us-female.mp3"></span>, *poor* <span class="pho alt">pʊr</span><span class="speak-word-inline" data-audio-us-male="/audios/poor-us-male.mp3" data-audio-us-female="/audios/poor-us-female.mp3"></span>, *your* <span class="pho alt">jʊr</span><span class="speak-word-inline" data-audio-us-male="/audios/your-us-male.mp3" data-audio-us-female="/audios/your-us-female.mp3"></span>…… 試過之後，再進行比較，你會發現更多的時候，第二種方式比第一種方式更方便。

在美式英語中，絕大多數人都使用第二種方式（*Bunched R*）發出兒化音。即便是使用第一種方式（*Retroflex R*），舌尖無論是前卷還是後卷，都一樣，絕不會貼到齦脊上。

另外，子音 <span class="pho">ʃ/ʒ</span>、<span class="pho">tʃ/dʒ</span> 在發音的時候，舌背也是同樣的位置。
