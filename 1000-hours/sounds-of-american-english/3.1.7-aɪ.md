## 3.1.7.2. 長度

亞洲語系基本上都沒有和英文一樣**長度**的**雙母音**。韓文日文的母音和中文的韻母雖然都有少數由兩個基礎音素構成的組合音，但，它們和英文的雙母音不同的地方在於，它們都是和其它韻母或者母音**等長**的，而不是像英文雙母音的長度那樣幾乎是短母音的**兩倍**。

比如，當中國人說 “太累了”<span class="speak-word-inline" data-audio-other="/audios/太累了-zh-cn-male.mp3"></span> 的時候，“太” <span class="pho">[tài]</span> 這個字的 “韻母” 就很接近英文的雙母音 <span class="pho">aɪ</span><span class="speak-word-inline" data-audio-us-male="/audios/us_phonetics_sound_eye_2023feb.mp3"></span>；“累” <span class="pho">[lèi]</span> 這個字的韻母就很接近英文的雙母音 <span class="pho">eɪ</span><span class="speak-word-inline" data-audio-us-male="/audios/us_phonetics_sound_day_2023feb_002.mp3"></span>…… 但，它們的長度就是不一樣，英文的雙母音總是相對更長。

英文的雙母音，都是兩個母音的組合。它們都是從第一個母音**滑**向第二個母音，所以，它們從長度上來看，相當於是長母音；**節奏**在感覺上大抵是樂譜裡的 “一拍裡均放著三個等長音符的三連音”，前一個音素更長一些，後一個音素相對較短。

讀 <span class="pho">aɪ</span><span class="speak-word-inline" data-audio-us-male="/audios/us_phonetics_sound_eye_2023feb.mp3"></span>, <span class="pho">eɪ</span><span class="speak-word-inline" data-audio-us-male="/audios/us_phonetics_sound_day_2023feb_002.mp3"></span>, <span class="pho">ɔɪ</span><span class="speak-word-inline" data-audio-us-male="/audios/us_phonetics_sound_boy_2023feb.mp3"></span> 的時候，唇形分別從 <span class="pho">ɑ</span><span class="speak-word-inline" data-audio-us-male="/audios/us_phonetics_sound_father_2023feb.mp3"></span> <span class="pho">e</span><span class="speak-word-inline" data-audio-us-male="/audios/us_phonetics_sound_head_2023feb.mp3"></span>, <span class="pho">ɔ</span><span class="speak-word-inline" data-audio-us-male="/audios/us_phonetics_sound_horse_2023feb.mp3"></span> 開始，唇形保持不變，用喉嚨完成從起始母音到 <span class="pho">ɪ</span><span class="speak-word-inline" data-audio-us-male="/audios/us_phonetics_sound_ship_2023feb.mp3"></span> 的過度。

![vowels-mouth-shape-aɪ](/images/vowels-mouth-shape-aɪ.svg)

讀 <span class="pho">aʊ</span><span class="speak-word-inline" data-audio-us-male="/audios/us_phonetics_sound_mouth_2023feb.mp3"></span> 和 <span class="pho">oʊ</span><span class="speak-word-inline" data-audio-us-male="/audios/us_phonetics_sound_nose_2023feb.mp3"></span> 的時候，唇形有變化，要分別從 <span class="pho">ɑ</span><span class="speak-word-inline" data-audio-us-male="/audios/us_phonetics_sound_father_2023feb.mp3"></span> 和 <span class="pho">ə</span><span class="speak-word-inline" data-audio-us-male="/audios/us_phonetics_sound_above_2023feb.mp3"></span> 開始，而後過度到 <span class="pho">ʊ</span><span class="speak-word-inline" data-audio-us-male="/audios/us_phonetics_sound_foot_2023feb.mp3"></span>。

![vowels-mouth-shape-aʊ](/images/vowels-mouth-shape-aʊ.svg)

同樣的道理，讀之前講過的兒化音 <span class="pho">er</span><span class="speak-word-inline" data-audio-us-male="/audios/us_phonetics_sound_hair_2023feb.mp3"></span>、<span class="pho">ɪr</span><span class="speak-word-inline" data-audio-us-male="/audios/us_phonetics_sound_ear_2023feb.mp3"></span>、<span class="pho">ʊr</span><span class="speak-word-inline" data-audio-us-male="/audios/us_phonetics_sound_pure_2023feb.mp3"></span> 的時候也一樣，唇形分別從 <span class="pho">e</span><span class="speak-word-inline" data-audio-us-male="/audios/us_phonetics_sound_head_2023feb.mp3"></span>、<span class="pho">i</span><span class="speak-word-inline" data-audio-us-male="/audios/us_phonetics_sound_sheep_2023feb.mp3"></span>（注意不是 <span class="pho">ɪ</span>）、<span class="pho">ʊ</span><span class="speak-word-inline" data-audio-us-male="/audios/us_phonetics_sound_foot_2023feb.mp3"></span> 開始，而後唇形保持不變，接著用喉嚨完成從起始母音到 <span class="pho">ə</span><span class="speak-word-inline" data-audio-us-male="/audios/us_phonetics_sound_above_2023feb.mp3"></span> 的過度 —— 而它的兒化音 <span class="pho">ɚ</span><span class="speak-word-inline" data-audio-us-male="/audios/us_phonetics_sound_mother_2023feb.mp3"></span> 則使用舌背後縮的方式。所以它們相當於是 <span class="pho">eɚ, ɪɚ, ʊɚ</span>，進而乾脆寫成了 <span class="pho">er, ɪr, ʊr</span>。

![vowels-mouth-shape-eə](/images/vowels-mouth-shape-eə.svg)

在讀它們的時候，最重要的是，要把它們讀得**足夠長** —— 幾乎要佔用兩個母音的發聲時長，並且足夠飽滿。

除了雙母音之外，還有另外兩個**三母音**：<span class="pho">aɪr</span><span class="speak-word-inline" data-audio-us-male="/audios/us_phonetics_sound_fire_2023feb.mp3"></span> 和 <span class="pho">aʊr</span><span class="speak-word-inline" data-audio-us-male="/audios/us_phonetics_sound_hour_2023feb.mp3"></span> —— 同樣要刻意拉長。
