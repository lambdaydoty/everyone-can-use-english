# 3.3. 變體

雖然幾乎每個權威詞典都聲稱自己採用的是 “國際音標”（IPA），可實際上，每個詞典的音標標註都各不相同，都是國際音標的一個**自定義子集**。

事實上，音標本身就是個**不完美**的標註系統，無論是英文的音標，還是中文的拼音（或者注音），抑或是其他語言的類似系統，都只是 “盡了全力卻終究在一些細節上無法做好” 的那麼一種存在 —— 哪怕是歷經多年增補修改的國際音標。

幸虧，各種詞典的音標不同主要體現在母音上 —— 子音的標註大抵上都是一樣的 —— 所以，掌握了某一個音標體系之後，稍微再掙扎一下，就可以看懂各種詞典的音標。

| 母音音標                      | 例詞     | 劍橋                             | 牛津                              | 朗文                              | 韋氏                              | 柯林斯                           |
| ----------------------------- | -------- | -------------------------------- | --------------------------------- | --------------------------------- | --------------------------------- | -------------------------------- |
| <span class="pho">ʌ</span>    | *but*    | <span class="pho">bʌt</span>     | <span class="pho">bət</span>      | <span class="pho">bʌt</span>      | <span class="pho">bət</span>      | <span class="pho">bʌt</span>     |
| <span class="pho">ɚ</span>    | *father* | <span class="pho">ˈfɑː.ðɚ</span> | <span class="pho">ˈfɑðər</span>   | <span class="pho">ˈfɑːðər</span>  | <span class="pho">ˈfä-thər</span> | <span class="pho">fɑːðəʳ</span>  |
| <span class="pho">ɝː</span>   | *worker* | <span class="pho">ˈwɝː.kɚ</span> | <span class="pho">ˈwəːrkər</span> | <span class="pho">ˈwɜːrkər</span> | <span class="pho">ˈwər-kər</span> | <span class="pho">wɜːʳkəʳ</span> |
| <span class="pho">ɒ/ɑː</span> | *dog*    | <span class="pho">dɑːɡ</span>    | <span class="pho">dɔɡ</span>      | <span class="pho">dɒːɡ</span>     | <span class="pho">ˈdäg</span>     | <span class="pho">dɔːg</span>    |
| <span class="pho">e</span>    | *bed*    | <span class="pho">bed</span>     | <span class="pho">bɛd</span>      | <span class="pho">bed</span>      | <span class="pho">bed</span>      | <span class="pho">bed</span>     |
| <span class="pho">er</span>   | *air*    | <span class="pho">er</span>      | <span class="pho">ɛr</span>       | <span class="pho">er</span>       | <span class="pho">er</span>       | <span class="pho">eəʳ</span>     |
| <span class="pho">ɪr</span>   | *ear*    | <span class="pho">ɪr</span>      | <span class="pho">ɪr</span>       | <span class="pho">ɪr</span>       | <span class="pho">ir</span>       | <span class="pho">ɪəʳ</span>     |
| <span class="pho">ʊr</span>   | *pure*   | <span class="pho">pjʊr</span>    | <span class="pho">pjʊr</span>     | <span class="pho">pjʊr</span>     | <span class="pho">ˈpyu̇r</span>    | <span class="pho">pjʊəʳ</span>   |
| <span class="pho">oʊ</span>   | *nose*   | <span class="pho">noʊz</span>    | <span class="pho">noʊz</span>     | <span class="pho">noʊz</span>     | <span class="pho">ˈnōz</span>     | <span class="pho">noʊz</span>    |

稍加整理，就會發現，各個詞典中最常不同的，變體最多的，其實只有 <span class="pho">ə</span>：

* 短母音版本 <span class="pho">ə</span>[^1]，在某些詞典中，可以用來表示 <span class="pho">ʌ</span> —— 比如，牛津詞典或韋氏詞典裡的 *but* <span class="pho alt">bət</span>；
* 短母音兒化音版本 <span class="pho">ɚ</span>, 可能被寫作 <span class="pho">ər</span> 或者 <span class="pho">əʳ</span>；進而，在柯林斯詞典裡，<span class="pho">ɪr</span> 被寫作 <span class="pho">ɪəʳ</span>，<span class="pho">er</span> 被寫作 <span class="pho">eəʳ</span>, <span class="pho">ʊr</span> 被寫作 <span class="pho">ʊəʳ</span>……
* 長母音 <span class="pho">ɝː</span>，可能被寫作 <span class="pho">əːr</span>，<span class="pho">ɜːr</span>，或者 <span class="pho">ɜːʳ</span>。

另外，<span class="pho">e</span> 有可能被寫成 <span class="pho">ɛ</span>，比如牛津詞典。

而 <span class="pho">ɒ/ɑː</span> 最混亂，各有各的寫法。比如 *dog*：

| 母音音標                      | 例詞  | 劍橋                          | 牛津                         | 朗文                          | 韋氏                          | 柯林斯                        |
| ----------------------------- | ----- | ----------------------------- | ---------------------------- | ----------------------------- | ----------------------------- | ----------------------------- |
| <span class="pho">ɒ/ɑː</span> | *dog* | <span class="pho">dɑːɡ</span> | <span class="pho">dɔɡ</span> | <span class="pho">dɒːɡ</span> | <span class="pho">ˈdäg</span> | <span class="pho">dɔːg</span> |

韋氏詞典的音標相對最特殊，可能需要專門適應[^2]。比如，雙母音使用上橫標表示：

| 劍橋 | <span class="pho">aɪ</span>  | <span class="pho">eɪ</span>  | <span class="pho">oʊ</span>  |
| ---- | --- | --- | --- |
| 韋氏 | <span class="pho">ɪ̅</span>   | <span class="pho">a̅</span>   | <span class="pho">o̅</span>   |

韋氏詞典的子音也相對比較獨特。比如，它用 <span class="pho">th</span> 標註 <span class="pho">θ</span> ，用帶下劃線的 <u><span class="pho">th</span></u> 標註 <span class="pho">ð</span>…… 初學者可以完全不必理會韋氏詞典的音標。

其實，快速掌握音標的最簡單直接有效的方法就是**勤查詞典** —— 看的次數越多，看的密度越高，就自然而然地熟悉了…… 不過，在一段時間裡，最好只查一個特定的詞典。

另外，在附錄 [8.2. 獲取 CEPD 音標](8.2-cepd-phonetics-and-sound)中，我介紹了一種可以在 macOS 上快速查詢並獲得《劍橋英語發聲詞典》中的音標和發音的方法。

事實上，我們還會遇到很多其它的變體。比如，在 Enjoy App 中，在底層對音訊檔案進行語音識別的引擎，用的是 espeak，[它就有一套自己的音標體系](https://github.com/espeak-ng/espeak-ng/blob/master/docs/phonemes.md)；而語音評估採用的引擎是微軟的 Azure，[它也有自己的一套音標體系](https://learn.microsoft.com/en-us/azure/ai-services/speech-service/speech-ssml-phonetic-sets#section)…… 而真正的解決方案其實是 “**態度上的選擇**” —— **不怕麻煩**。說實話，除此之外，根本就沒有其它的決方案。

[^1]: 比較罕見的情況下，短母音 <span class="pho">ə</span> 會被寫作 <span class="pho">ɜ</span>。
[^2]: https://merriam-webster.com/assets/mw/static/pdf/help/guide-to-pronunciation.pdf
