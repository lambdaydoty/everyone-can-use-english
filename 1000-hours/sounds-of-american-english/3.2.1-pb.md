# 3.2.1. <span class="pho">p/b</span>

<span class="pho">p/b</span> 的發聲動作一樣，都是先把嘴唇閉合，而後用氣流衝開嘴唇。<span class="pho">p</span> 沒有聲帶振動 —— 所以是清子音；而 <span class="pho">b</span> 有聲帶振動 —— 所以是濁子音。

## 3.2.1.2. 極輕母音 <span class="pho">ɤ</span>

不加講解的話，很多人會以為這兩個子音都會自然帶有很輕微的 <span class="pho">ʊ</span> 的音色 —— 是氣流衝開嘴唇自然產生的聲音。

> [!Note]
>
> 《劍橋英語發聲詞典》中，子音音素示例都刻意採用了 <span class="pho">ə</span> 的音色，例如，把 <span class="pho">p</span> 讀成 <span class="pho alt">pə</span><span class="speak-word-inline" data-audio-us-male="/audios/us_phonetics_sound_pen_2023feb.mp3"></span>，把 <span class="pho">b</span> 讀成 <span class="pho alt">bə</span><span class="speak-word-inline" data-audio-us-male="/audios/us_phonetics_sound_book_2023feb.mp3"></span>。

可事實上，那並不是 <span class="pho">ʊ</span>，而是另外一個很輕的母音，是一個普遍沒有被收錄到英文音標裡的國際音標符號，<span class="pho">ɤ</span><span class="speak-word-inline" data-audio-other="/audios/Close-mid_back_unrounded_vowel.ogg.mp3"></span>。其實，中文也有這個音，比如，在 “思念” 這個詞中， “思”（<span class="pho">sī</span>）的韻母就是這個音，另外，韓文裡也有這個音（韓文字元為 <span class="pho">ᅳ</span>），但日文裡沒有這個音。

雖然英文的音標列表裡並沒有收錄 <span class="pho">ɤ</span> 這個音，可實際上，你試試就知道了：<span class="pho">d</span> 常常被讀作 <span class="pho">dɤ</span>，<span class="pho">g</span> 常常被讀作 <span class="pho">gɤ</span> —— 其中的 <span class="pho">ɤ</span> 很輕很短；同樣，<span class="pho">t</span> 實際上被讀作 <span class="pho">tɤ</span>，<span class="pho">k</span> 被讀作 <span class="pho">kɤ</span>。

之前講過，非重音音節裡的母音可能會被弱化為 *schwa*（[3.1.1.2](3.1.1-ə#_3-1-1-2-央母音-ə)）<span class="pho">ə</span>…… 在自然語流中，連 *schwa* <span class="pho">ə</span> 都可能會被進一步弱化,變成非常輕的 <span class="pho">ɤ</span>。比如，常用詞 *reasonable*，字典裡標註的當然是 <span class="pho alt">ˈriːzənəbəl</span>，實際聽到的常常是 <span class="pho alt">ˈriːzɤnəbəl</span><span class="speak-word-inline" data-audio-us-female="/audios/reasonable-ɤ-us-female.mp3"></span> —— 拼寫成 *so* 的這個音節，發音可能並不是詞典裡標註的 <span class="pho alt">zə</span>，而是 <span class="pho alt">zɤ</span> 甚至乾脆只剩下 <span class="pho alt">z</span>……

## 3.2.1.3. 阻塞音 <span class="pho">◌̚</span>

有些子音 —— 比如，<span class="pho">p</span>, <span class="pho">g</span>, <span class="pho">t</span>, <span class="pho">θ</span> —— 處於**自然語流**末尾（比如句子末尾，或者換氣之前的意群末尾）的時候常常並不發音。這樣的時候，需要引入另外一個英語音標裡並未收錄的符號，<span class="pho">◌̚</span>。

這裡的 “不發音” 是指 “**聲帶沒有振動**” 但 “**其他發聲器官 —— 比如嘴唇、舌尖或喉嚨 —— 動作完整**”。

![articulator-tongue-tip-positions](/images/articulator-tongue-tip-positions.svg)

> * <span class="pho">p̚</span>，嘴唇閉上，而後並沒有繼續張開，所以沒有氣流輸出，也沒有隨後極其輕微的 <span class="pho">ɤ</span>。比如 *cup* <span class="pho alt">kʌp</span><span class="speak-word-inline" data-audio-us-male="/audios/cup-stop-us-male.mp3" data-audio-us-female="/audios/cup-stop-us-female.mp3"></span>
> * <span class="pho">g̚</span>，舌背後部抬起來頂住軟顎，而後並不離開，所以沒有氣流輸出，也沒有隨後極其輕微的 <span class="pho">ɤ</span>。比如 *big* <span class="pho alt">bɪg</span><span class="speak-word-inline" data-audio-us-male="/audios/big-stop-us-male.mp3" data-audio-us-female="/audios/big-stop-us-female.mp3"></span>
> * <span class="pho">t̚</span>，舌尖頂到位置 ③，而後並沒有離開，所以沒有氣流輸出，也沒有隨後極其輕微的 <span class="pho">ɤ</span>。比如 *but* <span class="pho alt">bʌt</span><span class="speak-word-inline" data-audio-us-male="/audios/but-stop-us-male.mp3" data-audio-us-female="/audios/but-stop-us-female.mp3"></span>
> * <span class="pho">θ̚</span>，舌尖頂到位置 ①，而後並沒有離開，所以沒有氣流輸出，也沒有隨後極其輕微的 <span class="pho">ɤ</span>。比如 *math* <span class="pho alt">mæθ</span><span class="speak-word-inline" data-audio-us-male="/audios/math-stop-us-male.mp3" data-audio-us-female="/audios/math-stop-us-female.mp3"></span>

這些阻塞音其實都有專門的名稱，我們會在後面的章節（[4.2.3. 頓/停頓](4.3-grouping#_4-3-2-停-暫停-pause)）給出另外一些句子中的例子。
