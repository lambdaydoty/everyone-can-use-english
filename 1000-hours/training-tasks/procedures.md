# 2. 訓練方法

請後續反覆認真閱讀《[語音塑造](/sounds-of-american-english/0-intro)》中的每一個章節。

## 2.1. 搞清每個單詞的讀音

在掙扎著**讀準**之前，**首先要避免讀錯**。這只是一個習慣，遇到任何不確定的詞彙都要查詞典 —— 當然，在有各種輔助工具的情況下（[GPT: Phonetic Transcriber](https://chat.openai.com/g/g-nWwuxUVqO-phonetic-transcriber)），甚至可以整句查詢音標。

望文生音非常可怕。我曾經在很多年裡把 *facade* 讀成 <span class="pho alt">ˈfækeɪd</span> —— 其實是 <span class="pho alt">fəˈsɑːd</span><span class="speak-word-inline" data-audio-us-male="/audios/facade-us-male.mp3" data-audio-us-femail="/audios/facade-us-female.mp3"></span>）；我也曾把 *specific* 讀成 <span class="pho alt">ˈspesɪfɪk</span> —— 其實是 <span class="pho alt">spəˈsɪfɪk</span><span class="speak-word-inline" data-audio-us-male="/audios/specific-us-male.mp3" data-audio-us-female="/audios/specific-us-female.mp3"></span>）…… 不妨再猜猜 *plaid* 這個詞怎麼讀？我在很長一段時間裡誤以為是 <span class="pho alt">pleɪd</span> …… 後來才發現並非如此 —— 其實是 <span class="pho alt">plæd</span> <span class="speak-word-inline" data-audio-us-male="/audios/plaid-us-male.mp3" data-audio-us-female="/audios/plaid-us-female.mp3"></span>。

如果不養成這個習慣，那麼讀錯的單詞會越來越多，等於時時刻刻在為自己的牢牆添磚加瓦，到最後再也無法逃脫 —— Github 上有個[常見中國程式設計師發音錯誤的單詞列表](https://github.com/shimohq/chinese-programmer-wrong-pronunciation)，不妨去看看，當作自己的警鐘。

遇到自己不熟悉發音的詞彙，要單獨反覆大聲朗讀，直至自己的舌頭和嘴唇都協調得非常輕鬆自然。每個人遇到的困難不一樣。比如，Instagram 上就有個人[演示](https://www.instagram.com/reel/C1VmSpuIvTO/?igsh=czY3cDAzcGIxMGN4)過自己 *literally* 這個單詞死活讀不明白的尷尬<span class="speak-word-inline" data-audio-uk="/audios/literally-instagram.mp3"></span> —— 不過，這恰恰就是在自己的大腦裡**新建連線新建網路**所要面臨的困難，要透過練習克服的就是這個困難，並且還要透過短時間內足量重複把**新連線新網路**強化到**原有近似連線近似網路**無法競爭的地步……

## 2.2. 放慢播放速度仔細聽

要慢速聽**很多遍**。從 2 大層面總計 7 個方面分別去分析，詳情參見《[語音塑造](/sounds-of-american-english/0-intro)》：

> * 韻律節奏
>   * 停頓（[4.2](/sounds-of-american-english/4.3-grouping#_4-2-1-意群)）
>   * 高低（[4.5.4](/sounds-of-american-english/4.5-sentences#_4-5-4-音高變化)）
>   * 起伏（[4.5.5](/sounds-of-american-english/4.5-sentences#_4-5-5-聲調變化)）
>   * 輕重（[4.5.3](/sounds-of-american-english/4.5-sentences#_4-5-3-強讀式與弱讀式的區別)）
>   * 緩急（[4.5.6](/sounds-of-american-english/4.5-sentences#_4-5-6-語速)）
> * 音素變化
>   * 母音（[3.1](/sounds-of-american-english/3.1-vowels)）
>   * 子音（[3.2](/sounds-of-american-english/3.2-consonants)）

**仔細聽**的意思是說，按照以上的順序，逐條分析，直至每個**音素**都認真分析過、認真體會過、認真嘗試過。這一步最需要**注意力集中**，也是整個訓練中最關鍵的部分 —— 這場訓練，從另外一個角度來看，實際上都是**注意力訓練**。

如果這一點沒做到、沒做好的話，就會出現 “**明明聽到的是一回事，腦子裡反應的卻是另外一回事**”。

比如，假設過去你一直習慣於把 *recent* <span class="pho alt">ˈrisənt</span> 裡的 <span class="pho alt">s</span> 讀成了 <span class="pho alt">z</span>，而後在這一步中你並沒有做到**仔細聽並分析到音素級別**的話…… 那麼接下來弄不好無論聽多少遍，無論跟讀多少遍，你都有可能依然**無法發現**你有一個音竟然讀錯了 —— 你的大腦會固執地用原有的連線和網路，乃至於自己毫無察覺。如果有人提醒，你會感覺特別奇怪，感覺上那麼**明顯**的差異，自己怎麼就**竟然**一直沒注意到？這裡有個示例，<span class="speak-word-inline" data-audio-other="/audios/recent-z.mp3"></span>（*recent health tips*）—— 這是某位同學被發現之前完全沒有意識到的錯誤。

在這一步，每個單詞都可以跟讀一下，透過試驗感覺自己的聲音，認真找自己需要調整的地方。

在最初尚不熟悉的時候，可以把文字和音標列印出來，在紙上用筆做各種標記。

> * 用 <span class="pho">|</span> 標註**可換氣停頓**，用 <span class="pho">·</span> 標註**非換氣停頓**；
> * 在**音高**（*pitch*）被拔高的音節上標註一個 <span class="pho">^</span>；
> * 在**重讀**的單詞可用加重字型作為標註；
> * 需要讀長的**母音**在下面或者上面畫一段直線；
> * 在必要的重讀音節的母音上標註**聲調**，<span class="pho">◌̅, ◌́, ◌̀</span>
> * **連讀**用 <span class="pho">◌‿◌</span> 符號做出標記
> * 在**讀得很快的幾個詞彙**下畫波浪線；
> * 格外需要注意並調整的**音素**，引出一個箭頭之後寫上各種必要的筆記……

這一步的仔細分析很重要，是後面所有練習步驟的必要前提。請注意，這一步分析，弄上半小時甚至一小時都不過分…… 因為**你不知道的你就注意不到**。

## 2.3. 分段跟讀

這和我們練琴特別像。剛開始的時候，人家彈得輕鬆聽著優美，我們自己的手指乾脆不聽話，笨得跟腳趾似的……

<video controls><source src="/videos/hc-gt-normal.mp4"></source></video>

於是，只能放慢聽，放慢速度練……

<video controls><source src="/videos/hc-gt-slow.mp4"></source></video>

 可即便放慢速度了也不行…… 於是，只能一小節一小節地練，練熟了一小節再去練下一小節，兩個小節都練熟了之後吧，還得拼起來重複很多遍才能做到兩個小節之間銜接自然…… 如此這般，一小節一小節地終於 “**爬**” 完之後，才算有資格可以進行**正式練習**了……

自然語音中有**暫停**（*pause*）的地方，就相當於是樂譜裡的小節分界線。分段練習，實在難的小節，還可以繼續拆分……

在跟讀的時候，有必要**誇張**一點。聲音也要正常地**大** —— 確實不用喊，嗓子的確需要保護。實際上，我們主要需要練的是嘴唇、舌頭、氣流振動的配合，主要練的還真不是聲帶。

跟讀的時候，可以戴著耳機。一個比較好的方法是戴單隻耳機 —— 這樣，一方面錄音聽得更清楚，另外一方面也不妨礙聽清自己的聲音，還可以換著耳朵戴。

上一步中的**音素級分析**，在這一步中開始發揮作用，因為你可以注意到每一個細節。而後的過程，漫長且又難過 —— **短時間內足量重複** —— 要在大腦裡建立新連線強化新連線，建立新網路強化新網路，防止舊連線舊網路被啟用，防止近似連線近似網路劫持操作…… 時時刻刻感覺到自己的笨拙，苦惱於做不到、做不好 —— **非常難過，但必需忍受**。

不要害怕自己的笨拙。

大家都一樣，剛開始的時候就是很吃力，就是很生硬 —— 並且，這種笨拙必然要持續相當長一段時間…… 哪怕感覺上已經熟練，還是需要繼續練習的 —— 做什麼都一樣。

TED 上有一個講座，Benjamin Zander 講 [The transformative power of classical music](https://www.ted.com/talks/benjamin_zander_the_transformative_power_of_classical_music/transcript?language=en)，提到說 “沒有人是音盲（*tune deaf*）” 所以每個人都可以練習演奏 —— 這很重要…… 不過，值得你在這裡聽聽的是他模仿了不同時期孩子彈鋼琴的情況，從生硬到熟練……

<audio controls><source src="/audios/benjamin-zander-on-kids-playing-piano.mp3"></source></audio>

其實，無論幹什麼，每個人都得經歷那其中的各個階段。

## 2.4. 完整跟讀

終於，可以嘗試**完整跟讀**了。

在這個階段裡，千萬不能誤以為自己現在已經很熟練了就徹底過關了。

完整跟讀幾遍之後，要錄音。這時候，之前做的詳細的標記就更起作用了。你會發現自己總是會在一些想象不到的地方一不小心就變成了過往自己說話的樣子 —— 這就是近似連線近似網路和新建連線新建網路競爭勝出的結果。也許你需要換個顏色的筆再做一些**重點標記**。

重複三五遍之後，就要再錄一次音，再對比再調整。

**錄音非常重要**。因為不錄音的話，很多自己沒弄明白的細節就被掩蓋了。只要錄音，沒做好沒做對的地方就無可遁形。倒不見得每一次跟讀都要錄音，但，一定要時不時錄一次。

## 2.5. 背誦

**這一步格外重要，絕不能糊弄！**—— 這是兩個核心任務之一。隨著時間的推移，**記憶擴充套件**的重要性會逐步超過**語音塑造** —— 畢竟，**語音塑造**練一段時間之後就可以用一輩子；但，**記憶擴充套件**卻是可以且最好長期持續進行的。

切記：**記憶是一切語言能力的基礎**。

這時候，你已經把整個文字讀過不知道多少遍了 —— 背誦不應該有太多困難。

然而，由於它並不是用你目前高度熟悉的語言書寫的，所以你還真不一定能夠一下子記住。要集中注意力短時間內重複多次才可以。

剛開始的時候可能會感覺死活記不住…… 放心，**你的記憶力會越來越好的，只要你肯練**。

每隔一段時間要專門騰出一天覆習之前的內容。比如，週日的時候，專門複習這一整週的所有內容。同樣，到了月末，要騰出兩三天專門複習這一整個月的內容。

## 2.6. 放鬆

在此之前，你的所有努力都是要儘量用錄音裡的方式說出所有的文字。在這個過程中，為了保證練習效果，你可能一直相當誇張，這也非常必要。

在練習的過程中，絕大多數人都會不由自主受到的影響有一部分來自於示範錄音中的**音質**。

無論是人工智慧生成的所謂真人語音，還是有聲書裡的錄音，抑或是影視劇裡的臺詞 —— 這些錄音中的朗讀者**音質**也好**聲線**也罷，都是訓練有素的**表演**。絕大多數人在日常生活裡並未受到過這方面的專業訓練，肯定不影響日常交流，但，的確很難做到在音質上或者聲線上模仿。

當你熟悉到一定程度之後，你可以嘗試著放鬆自己的聲音。這樣多少可以擺脫音質或聲線造成的影響。但，依然需要重點關注的是那兩大方面：**韻律節奏**和**音素變化** —— 這些都與音質或聲線無關。

放鬆自己的嘴唇、舌頭、呼吸，用盡量自然的方式反覆朗讀文字。重複幾遍之後就錄一次音…… 對比一下…… 也需要做調整…… 嘗試背誦…… —— 重複以上步驟。

## 2.7. 覆盤

一天至少三個小時，每週最多可以中止一天。

每天結束之前，要花 5～10 分鐘做一下**覆盤**，回憶並記錄一下自己今天遇到的困難、已經克服的困難、尚未克服的困難、以及面對那些困難時所採用的方法…… 想一想還有什麼方法可以試試？這些都要寫下來。相信我，記錄這個東西 ，總是會以想象不到的方式起想象不到的作用。
