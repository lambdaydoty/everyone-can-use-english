# 1. 基礎語言能力

你有沒有認真想過，我們經常提起或者偶爾稱讚的**文筆**或者**口才**指的到底是什麼呢？

文字是用來記錄並傳遞感知的。所以，一個人能用的名詞，往往侷限在他能夠感知的事物的總和之中，不管是抽象的還是具體的。他們能用的動詞，就是這個集合裡的事物能夠發出或者承載的動作。

**具體事物**，見過就知道，沒見過就不知道也很容易理解。能用語言描述具體的事物及其對他們的感知，可以算作是**基礎語言能力**了吧？

這基礎之上的才可以被稱為**文采**或者**口才**吧？

毫無疑問，**抽象事物**的數量一直在不斷增加，並且發展得越來越快。亞里士多德顯然是個天才，但他不知道**重力**的存在；發明瞭望遠鏡的伽利略當然好奇過天上到底有多少顆星星，可天才如他也無法想象許多年後有科學家想到了個用照片影象上的**景深**去估算恆星數量的方法；伽利略肯定也沒聽說過**進化**這個概念；達爾文不可能知道什麼是**量子糾纏**或者**指數基金**。

“幸福” 這樣普遍的抽象概念雖然理解上並不統一，但大家好像早都知道，也不需要教育。但更多的抽象概念，比如，**適者生存**，**雙盲測試**，**舉證責任**，**基尼係數**，**皮格馬利翁效應**，**梅特卡夫定律**，**摩爾定律**，**聚光燈效應**，**貝葉斯定理**…… 這些今天的人們可能習以為常的抽象概念，往往需要數年的教育或者自我教育才可以真正習得並熟練應用。

當人們討論語言中的修辭現象的時候，會膚淺地把它劃分到語言能力之中。可本質上來看，精彩且又準確的修辭，更多基於敏銳的觀察和豐富且深入的思考。

先不說那些高階的**修辭**，單說對一個事物的**形容**，即，普通的形容詞，人與人之間的差異也非常大。

如果一個人對一個他能夠感知到的事物只有一方面的感知，那麼針對這個事物，他可用的形容詞也就只有一個，不可能是多個。比如，同樣一部電視劇，很多人只有一個感受，“好看”；而另外一個人的描述卻是，“這是一個畫面華麗，情緒充沛，卻故事單薄，人物拉垮，但反過來又因為這個時代的品味而大火特火的電視劇”。與其說這是文采，不如說這只是對同一事物更多角度的觀察、感知與思考最終在語言上的體現。

有人喜歡**押韻**，有人善用**類比**，有人擅用**隱喻**，有人以**排比**為樂，有人以**旁徵博引**為榮。

**押韻**雖然看起來更多是語言能力，但，不可否認的是，它來自更多的練習，甚至鑽研，以及長期的積累。更為底層的，是足夠的記憶力 —— 首先要有足夠量的選擇，然後才能在眾多的選擇中找到恰當的韻腳。比如，“只要掌握了這項能力，你就無往不利…… 所向披靡。”

一個人能用精確且又出人意料的**類比**，說明他能看到別人看不到的不同事物之間的關聯，並且還能在眾多的關聯裡挑出最準確最令人震驚的那個…… 這可是非常了不起甚至罕見的技能。

一個人能不動聲色地用好各種**隱喻**，往往來自整個文化的長期影響。比如，同樣說，“他就是一顆**棋子**” — 這倒真是全球通用的一個隱喻，畢竟，**人生如棋**…… 但，接下來，這個 “棋子” 到底是什麼呢？歐洲人可能在 *Knight*, *Bishop* 和 *Pawn* 中選擇（國際象棋）；而中國人呢？車馬炮，或者卒（中國象棋）。

至於**排比**和**旁徵博引**麼，毫無疑問，都需要更多額外的功課，因為現編總是來不及的。

這些普遍被認為是或者屬於文采或者口才，但，實際上，它們都不是**基礎語言能力**，它們都是另外的技能，幾乎各個都有各自獨立且有系統的知識體系。

再比如，邏輯嚴謹，這更是一項終生的修煉。邏輯是一種既簡單又困難的思考工具。你說它簡單吧，它也的確就那麼幾十個概念。日常討論中，無非是 “在 ‘因為’ 後面說清楚充分且必要的理由和根據”，或者 “在 ‘比如’ 後面給出精彩且又準確越多越好的例子”，看起來的確簡單。你說它困難吧，還真是，就算練上一輩子也隨時有可能突然之間陰溝裡翻船。

又比如，講故事的方式，這簡直是一門藝術。一切的事物都在一個網路裡不斷互動，描述它們語言竟然只有樹狀的語法，然後要想盡一切辦法讓對方線性地理解。先說什麼後說什麼，哪裡要插敘，哪裡要倒敘，哪裡要抖包袱，要怎樣才能做到情理之中卻又意料之外……

這些，還是跟基礎語言能力全無關係。

要非說說清楚 “基礎語言能力” 指的到底是什麼，大抵上可能包括以下幾個方面：

> * 足夠的詞彙量
> * 足夠的語法知識
> * 符合文化的通用表達方式

—— 僅此而已。至於以上提到的種種技巧或者能力，實際上都是基礎語言能力之外的東西，但它們有著同樣的特徵，都是隻能靠花時間和精力去積累才能發展出來的能力。
